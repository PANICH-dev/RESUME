Technical Highlight

Advanced Data Engineering Logic

พัฒนา Data Processing Engine ด้วยฟังก์ชันระดับสูง (Advanced Array Formula) เพื่อจัดการข้อมูลหลายมิติแบบ Real-time ทำหน้าที่คล้ายกระบวนการ ETL

  • ⚡ VSTACK & CHOOSECOLS: ผสานและดึงข้อมูลเฉพาะคอลัมน์เป้าหมายจากฐานข้อมูลหลักนับพันบรรทัด
  • ⚡ Dynamic LET Variables: ประกาศตัวแปรคำนวณพื้นที่ (Sq.Ft.) และจับคู่โครงสร้างราคา (VLOOKUP) ภายในสูตรเดียว
  • ⚡ Array Computing: คืนค่าผลลัพธ์เป็นตารางข้อมูล (Table Array) อัตโนมัติ โดยไม่ต้องพึ่งพา Script หน่วงระบบ
=ARRAYFORMULA(LET(
  raw, VSTACK(FILTER('ฐานข้อมูล'!A1:DH1000, 'ฐานข้อมูล'!A1:A1000<>""), ...),
  row_idx, SEQUENCE(ROWS(raw)),
  is_h, row_idx=1,

  colA, CHOOSECOLS(raw,1), colD, CHOOSECOLS(raw,4), ...

  // คำนวณพื้นที่ (Square Footage) แบบ Dynamic
  sq_front_val, IF(colAE="", 0, (colAG*colAH*colAI)/144),
  sq_rear_val, IF(colAM="", 0, (colAO*colAP*colAQ)/144),
  ...
  // Mapping ชื่อคอลัมน์อัตโนมัติ
  sq_front, IF(is_h, "ตร.ฟ.หน้า", IF(colAE="", "", sq_front_val)),
  ...
  // คำนวณต้นทุน Real-time ด้วย Array VLOOKUP
  c_front, IF(is_h, "ต้นทุนหน้า", IF(colAE="", 0, IFERROR(VLOOKUP(colAE, 'ราคาฟิล์ม'!A:D, 4, 0) * sq_front_val, 0))),
  ...
  // สร้าง Table Result สุดท้าย
  table, {
    colA, colD, colF, colG, ...
    colAE, sq_front, colAK, colAL, c_front,
    colAM, sq_rear, colAS, colAT, c_rear,
    ...
    colDH, c_total
  },

  condition, IFERROR((MONTH(colD) = MONTH(A3)) * (YEAR(colD) = YEAR(A3)), 0) + is_h,
  FILTER(table, condition)
))
